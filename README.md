# MEMFY for GIF

Полнофункциональный веб-редактор GIF в браузере. Загружайте GIF, редактируйте все кадры анимации, добавляйте текст, обрезайте, применяйте эффекты и экспортируйте новый GIF. Работает полностью на клиенте без сервера.

## Функционал

- **Загрузка GIF**: Drag-and-drop или кнопка. Анимация разбирается на кадры с сохранением задержек.
- **Отображение**: Первый кадр отображается на холсте для редактирования.
- **Редактирование**:
  - **Текст**: Добавление редактируемого текста (fabric.IText).
  - **Обрезка**: Выделение области и обрезка всех кадров.
  - **Эффекты**: Чёрно-белый фильтр, применяемый ко всем кадрам.
- **Экспорт**: Сборка отредактированных кадров в новый GIF с автоматической загрузкой.
- Все изменения синхронизируются между кадрами для сохранения анимации.

## Как использовать

1. Загрузите GIF.
2. Добавьте текст, обрежьте или примените эффекты.
3. Экспортируйте результат.

## Технологии

- Fabric.js для холста и объектов.
- omggif для парсинга GIF.
- GIF.js для сборки GIF.
- HTML5 Canvas.
